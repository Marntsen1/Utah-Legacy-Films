# Create GitHub Repository - Quick Steps

## Step 1: Create Repository on GitHub

1. **Go to**: https://github.com/new
2. **Repository name**: `utah-legacy-films`
3. **Description** (optional): "Cinematic family legacy video production website"
4. **Visibility**: Choose Public or Private
5. **IMPORTANT**: Don't check any boxes (no README, .gitignore, or license)
6. Click **"Create repository"**

## Step 2: Push Your Code

After creating the repository, come back here and I'll push the code for you!

Or run this command yourself:
```bash
git push -u origin main
```

You'll be prompted for your GitHub username and password/token.
